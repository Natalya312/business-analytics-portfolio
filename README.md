# Business Analytics Portfolio

This portfolio contains practical business analytics projects focused on logistics, shipments, process optimization and data analysis.

## Projects

### 1. Shipment Performance Analysis

**Tools:** SQL, Excel / Power BI / Tableau
**Area:** Logistics, shipments, delivery performance

**Project goal:**
Analyze shipment data to identify delivery delays, transportation costs, carrier performance and operational efficiency.

**Business questions:**

* What is the average delivery time?
* Which shipments were delivered late?
* What is the on-time delivery rate?
* Which carriers have the highest shipment costs?
* Which countries generate the highest logistics costs?
* What is the revenue and profit by carrier?
* Which customers generate the highest shipment volume?

**Files:**

* `shipments_data.csv` - shipment dataset
* `shipments_analysis.sql` - SQL queries for analysis
* `dashboard.png` - dashboard screenshot
