# Shipment Performance Analysis

## Project Overview

This project analyzes shipment performance data to identify delivery delays, carrier efficiency, route performance, and logistics bottlenecks.

## Tools Used

- SQL
- Excel
- GitHub

## Dataset

Source: Synthetic logistics dataset

Records:  5+ million shipment records

## Business Questions

- What is the on-time delivery rate?
- Which carriers perform best?
- Which routes experience the most delays?
- What are the main delay reasons?
