# Retail Sales Performance Analysis

## Project Overview

This project analyzes retail sales performance to identify factors influencing revenue, customer behavior, pricing strategy, inventory levels, and promotional effectiveness.

## Tools Used

* SQL
* Excel
* GitHub

## Dataset

Source: Retail Store Product Sales Dataset

Records: 15,000

## Business Questions

* How do discounts affect sales performance?
* What is the impact of advertising spend on store traffic?
* How does stock level influence revenue?
* How do competitor prices affect sales?
* What factors drive customer returns?
* Which variables have the strongest impact on business performance?
